# CBM_Flash_ROMs 
**Various Commodore Cartridge and ROM replacement projects.**

## 16BitFlashROM_Adaptor
Simple adaptor to use a 48 pin SMD flash rom in place of the AMIGA 500 dip rom.
Extra address pins on top to allow selection from multiple rom images.
[link to ibom](ibom_16BitFlashROM_Adaptor.html)

![16 Bit Flash Rom Adaptor](16BitFlashROM_Adaptor/Images/16BitRomAdaptor.jpg?raw=true "16 Bit Flash Rom Adaptor")

## C64_FlashCart
Flash cartridge for the C64

## FlashCartProgrammer
Flash Cart Programmer with sockets for VIC20 & C64 cartridges plus pin header for programming
of my 16Bit Flash ROM Adaptor board.
[link to ibom](ibom_FlashCartProgrammer.html)

## FlashSPI
Raspberry PI Pico code used to program the SPI Flash ROM on my FPGA boards.

## Vic20_DiagROM
Dissassembly of vc-20-diag.324173-01_blk5.bin with partial comments.

## Vic20_FlashCart
Vic20 Ram Expansion and Multi ROM Image Flash Cartridge.
[link to ibom](ibom_Vic20_FlashCart.html)
